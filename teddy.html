Mean Teddy

A simple AI that talks meanly.

This app detects string inputs from the users and responses to the keywords in the strings. If there is no keywords found in the users' line, the AI would reply with some general lines.

Initial screen of the app, inviting player to interact with Teddy.

Talk to Teddy, and it asks for user's name. (the name will be saved as a String for later use in reponses.)

Teddy would try to ask questions to carry on the conversation.
